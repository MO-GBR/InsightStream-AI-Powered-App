# ✨ InsightStream | AI-Powered Social Listening & Brand Sentiment Dashboard.

- Instead of a basic chatbot, you'll build a platform where users input a brand name or keyword.
- The app fetches real-time mentions (simulated or via API), uses AI to analyze the "vibe" (sentiment), and provides actionable business advice.

## 🛠 The Tech Stack
- Frontend 👉 Next.js 16 (App Router), TypeScript, Tailwind CSS.
- Animations 👉 GSAP (for data entry/exit and smooth dashboard transitions).
- Backend 👉 Next.js Route Handlers
- Database 👉 PostgreSQL with Supabase
- AI/LLM 👉 Gemini 3 Flash + LangChain (for structured data extraction).
- Real-Time 👉 Supabase Subscriptions
- Auth 👉 Supabase Auth
