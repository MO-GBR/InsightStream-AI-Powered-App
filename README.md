# ✨ InsightStream

### AI-Powered Social Listening & Brand Sentiment Dashboard

InsightStream is a modern AI-driven platform that helps you monitor brand mentions, analyze sentiment, and stay ahead of potential crises in real time. It combines social listening, AI insights, and intelligent automation into a single clean dashboard.

---

## 🚀 Overview

InsightStream allows users to track keywords across the web (Reddit & RSS sources), analyze public sentiment using AI, and receive actionable insights through visual dashboards and daily voice briefings.

Whether you're a founder, marketer, or indie hacker, InsightStream helps you understand what people are saying about your brand — instantly.

---

## 🧠 Core Features

### 🔍 Social Listening Engine

* Track keywords across:

  * Reddit
  * RSS feeds
* Automatically collect and store mentions in real time

### 📊 AI Sentiment Analysis

* AI analyzes mentions and extracts:

  * Sentiment score (positive / neutral / negative)
  * Trends and patterns
  * Key discussion topics
* Data is visualized in charts and insights dashboards

### 🚨 Crisis Shield

* Detects sudden drops in sentiment
* Triggers alerts when risk is high
* Helps you react before issues escalate

### 💬 Smart AI Chat (RAG System)

* Upload files (PDFs, documents)
* Chat with your data using AI
* Perfect for internal knowledge or research

### 🎧 Daily Audio Briefing

* AI-generated summary of your mentions
* Delivered as natural human-like voice audio
* Quick way to stay updated without reading dashboards

### 🧹 Automated Data Cleanup

* Background worker removes very old data
* Keeps database clean and efficient
* Prevents unnecessary storage costs

## ⚙️ How It Works

1. Create a project with a keyword
2. InsightStream starts collecting mentions from Reddit & RSS
3. AI processes and analyzes sentiment
4. Data is displayed in dashboards & charts
5. Alerts trigger if sentiment becomes risky
6. Receive a daily AI-generated audio summary
